RDruid
======

[Druid](https://github.com/metamx/druid/wiki) connector for R

## Quick Start

To install the latest version of RDruid, it's easiest to use the `devtools` package:

```r
install.packages("devtools")
devtools::install_github("RDruid", "metamx")
```
